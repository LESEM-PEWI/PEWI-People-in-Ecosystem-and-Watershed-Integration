PEWI offers a Print Function, where you can print your maps, results, and much more. Activate the Print function by pressing "P" on the keyboard and select what you want to be printed out on a PDF. You can select from a number of categories to choose from to print, including: Landuse Type Maps for set years, results, levels maps, features maps, and yield maps. There are features such as "Select All" and "Reset" to help in the customization process. Once finished, click "Preview PDF" to see what will be on your PDF and then "Download" when ready. Exit when finished.
